## 💻 Getting started

**Clone the project and access the folder**

```bash
$ git clone https://github.com/LucieneFreitas/github-blog.git && cd github-blog
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm i

# To finish, run
$ npm run dev

# Well done, project is started!
```
